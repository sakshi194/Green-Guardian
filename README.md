# 🌿 Plant Store Website – Built Using Frappe Framework

This project is a custom website developed using the **Frappe Website Module**, designed for selling **plants and plant-accessory tools**.
It includes fully dynamic pages, product listings, and clean UI built through custom Doctypes and templating.

---

## 🚀 Features

* 🌱 Product listing for plants & accessories
* 📦 Dynamic data fetching using custom Doctypes
* 🖼️ Product images displayed directly from DocTypes
* 🧩 Custom app structure for scalability
* 🎨 Frontend built with Jinja, HTML, CSS & JavaScript
* 🐍 Backend logic in Python
* 🛒 eCommerce-style layout and user flow

---

## 🛠️ Tech Stack

* **Frappe Framework**
* **Python**
* **Jinja**
* **JavaScript**
* **HTML/CSS**
* **Frappe Website Module**
